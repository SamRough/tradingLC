# 香港股票交易流程可视化

一个使用 Emoji 可视化演示香港股票交易完整流程的交互式网页应用。

![Preview](preview.png)

## 功能特点

- **10个交易阶段完整演示**：从下单到交割的完整流程
- **交互式动画**：Emoji 动画展示订单、股票和资金的流动
- **买卖双方视角**：同时展示买方和卖方的操作流程
- **时间线控制**：支持播放、暂停、上一步、下一步、重置
- **可调节速度**：支持 1x 和 2x 播放速度

## 交易阶段

1. **下单** - 投资者向券商提交订单
2. **验证** - 券商进行风控检查、资金验证
3. **路由** - 订单发送到交易所
4. **撮合** - 交易所匹配买卖订单
5. **成交** - 成交信息返回
6. **清算** - HKSCC 作为中央对手方发出 T+2 清算指令
7. **资金** - T+2 资金清算：券商向银行发扣款指令，资金经清算系统划转后到账
8. **证券** - T+2 存管机构完成证券划转
9. **持仓** - 券商更新投资者账户持仓
10. **完成** - T+2 交割完成

## 参与实体

| 角色 | 说明 |
|------|------|
| 👤 买方/卖方投资者 | 买入或卖出股票 |
| 🏢 买方/卖方券商 | 交易通道 |
| 💳 买方/卖方银行 | 资金托管 |
| 📊 交易所 | 撮合成交 |
| 🏦 存管机构 | 证券托管 |

## 资金管理

### 入金流程

香港券商客户需要入金后才能买卖港股。入金方式包括：

- **FPS 转账** - 快速支付系统，最快到账
- **银行转账** - 通过网银/手机银行转账到券商银行账户
- **支票存入** - 存入券商指定银行账户

### 资金存放

| 层级 | 存放位置 | 说明 |
|-----|---------|------|
| 客户层 | 券商保证金账户 | 客户资金与券商自有资金隔离托管 |
| 券商层 | 券商银行账户 | 券商在银行开立的独立信托账户 |

根据香港证监会规定，客户资金必须存放在**独立信托账户**中，券商不得挪用，即使券商破产客户资金也受到保护。

### T+2 资金流转

```
入金 → 券商银行账户（托管）→ 清算系统划转 → 卖方券商 → 卖方投资者
```

## 技术栈

- 纯 HTML/CSS/JavaScript
- 无外部依赖
- 响应式设计，支持移动端

## 使用方法

直接在浏览器中打开 `index.html` 即可使用：

- 点击播放按钮开始演示
- 使用键盘空格键播放/暂停
- 使用左右箭头键切换阶段
- 点击时间线上的数字跳转到指定阶段
